# WaqfWise Frontend

Modern web application for managing Waqf (Islamic endowment) campaigns and donations.

## Tech Stack

### Core Libraries
- React 18 with TypeScript (type safety & better DX)
- React Router v7 (latest, data router with loaders & actions)
- Material UI v6 (comprehensive component library)
- React Query (tanstack-query) for data fetching & caching
- Zustand or Redux Toolkit for global state management
- React Hook Form + Zod for form validation
- Axios for HTTP client with interceptors
- i18next for internationalization (Indonesia + English + Arabic)

### Build Tools
- Vite (lightning-fast dev server & build)
- TypeScript 5.x
- ESLint + Prettier for code quality
- Vitest for unit testing
- Playwright for E2E testing

### Styling
- Material UI's sx prop & styled() API
- Custom theme with createTheme
- Emotion for CSS-in-JS (built-in MUI v5+)

## Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Test

```bash
npm run test
```
